COMO EMPEZAR CON GEV ICONS

1.- Copiar la dirección y pegar en el head de tu página.

http://google.com

2.- Colocar la clase del icono en tu elemento HTML.

<div class="gev-grafica">
....
</div>

También puedes agregar el código UNICODE a tu HTML e indicar en tu CSS el tipo de fuente

CSS

.custom-class{
  font-family: 'gevicons';
}

HTML

<div class="custom-class"> &#xf015; </div>
